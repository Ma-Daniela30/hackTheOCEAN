¿Qué hacemos?


Por medio de una API los usuarios podrán visualizar en tiempo real las zonas con mayor índice de contaminación de plástico en el medio. Esta API será utilizada para las organizaciones que se encargan de realizar campañas de limpieza en océanos.

Algunas de ellas podrian ser:

  • Red de Ecuadores Ambientales(REA)
  • Ocean Conservancy
